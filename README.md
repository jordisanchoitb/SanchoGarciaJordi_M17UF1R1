# SanchoGarciaJordi_M17UF1R1
VVVVV Game

## Instruccions de joc

### Tecles d’inputs per jugar
- **A**: Moure's a l'esquerra
- **D**: Moure's a la dreta
- **Espai**: Camvi de gravetat
- **Escape**: Pausar el joc

### Descripció del joc
**Gènere**: Plataformes

**Visualització**: Lateral
